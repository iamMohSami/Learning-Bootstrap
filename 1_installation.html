There are two main methods for installing Bootstrap: using a CDN (Content Delivery Network) or downloading the Bootstrap files.

A. Using CDN (Content Delivery Network)

A CDN is a network of servers distributed globally to deliver web content efficiently. Bootstrap can be easily integrated into your project by linking to the Bootstrap CDN in your HTML file.
METHOD : link bootstrap.min.css {CSS} and bootstrap.bundle.min.s {JS} which is a bundled version that includes Bootstrap's JavaScript, jQuery, and Popper.js in a single file.

Downloading Bootstrap

You can also download the Bootstrap files from getbootstrap.com and host them locally in your project. This method provides more control over versioning and can be useful in offline environments.
