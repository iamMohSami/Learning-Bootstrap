Bootstrap Containers: .container and .container-fluid

Containers are used to pad and center the content inside of them, providing a neat and responsive layout. There are two container classes available:

  • The .container class provides a responsive fixed-width container.
  • The .container-fluid class provides a full-width container, spanning the entire width of the viewport.
